# 코드 예시
```java
// 클래스: 설계도
class Car {
    String color; // 정의된 필드
   
    // 정의된 메서드
    void drive() {
        System.out.println("출력");
    }
}

public class Main {
    public static void main(String[] args) {
        // 객체 생성 (Car의 인스턴스)
        Car car = new Car();
        car.color = "Red"; // 정의된 필드 사용
        car.drive(); // 정의된 메서드 사용
    }
}
```
